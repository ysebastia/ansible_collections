## Requirements

- Minimum Ansible version: `2.1`

## Default Variables

### ansible_action

#### Default value

```YAML
ansible_action: config
```

### enable_pki_deploy_ca

#### Default value

```YAML
enable_pki_deploy_ca: false
```

### pki_ca_file

#### Default value

```YAML
pki_ca_file: ca-{{ pki_ca_name }}.pem
```

### pki_ca_group

#### Default value

```YAML
pki_ca_group: root
```

### pki_ca_host

#### Default value

```YAML
pki_ca_host: '{{ inventory_hostname_short }}'
```

### pki_ca_name

#### Default value

```YAML
pki_ca_name: ansible
```

### pki_ca_owner

#### Default value

```YAML
pki_ca_owner: root
```
